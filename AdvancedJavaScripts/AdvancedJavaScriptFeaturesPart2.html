6. Optional Chaining

Optional chaining ?. allows you to safely access deeply nested object properties without having to check if each level exists.
Real-life Example: Useful when accessing user profile or configuration data that may not be fully available. 

const user = { profile: { name: 'Bob' } };
console.log(user?.profile?.name); // "Bob"
console.log(user?.address?.street); // undefined

7. Nullish Coalescing Operator

The ?? operator returns the right-hand operand only when the left-hand is null or undefined (not 0, false, or '').
Real-life Example: Use for assigning default values only when data is truly missing. 

let name = null;
console.log(name ?? 'Default'); // "Default"

let count = 0;
console.log(count ?? 10); // 0

8. Generators

Generator functions (function*) return an iterator object and allow pausing/resuming execution using yield.
Real-life Example: Used in pagination, animation frames, or asynchronous workflows. 

function* generator() {
  yield 1;
  yield 2;
  yield 3;
}

const it = generator();
console.log(it.next().value); // 1
console.log(it.next().value); // 2

9. Proxy

A Proxy allows you to intercept and redefine fundamental operations (e.g., property lookup, assignment, enumeration).
Real-life Example: Used for validation, access control, or automatic property fallback. 

const user = { name: 'John' };
const proxy = new Proxy(user, {
  get(target, prop) {
    return prop in target ? target[prop] : 'Not Found';
  },
});

console.log(proxy.name); // "John"
console.log(proxy.age);  // "Not Found"

10. Symbols

Symbol creates a unique identifier, ideal for hiding object properties to avoid naming conflicts.
Real-life Example: Used in frameworks to define private object properties or constants. 

const id = Symbol('id');
const user = { [id]: 123 };
console.log(user[id]); // 123
